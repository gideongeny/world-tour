# World Tour - Travel Planning Application

A modern, world-class travel planning application built with React, Flask, and powered by Google's Gemini AI.

## Features
- Interactive map-based destination browsing
- AI-powered travel recommendations
- Real-time weather information
- Secure booking system
- Modern, responsive UI

## Tech Stack
- **Frontend**: React + Vite + Tailwind CSS v3
- **Backend**: Flask with Blueprints
- **AI**: Google Gemini API
- **Deployment**: Vercel (Frontend) + Google Cloud Run (Backend)

## Local Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
pip install -r requirements.txt
python app.py
```

## Deployment
The application is configured for automatic deployment via Vercel and GitHub Actions.
